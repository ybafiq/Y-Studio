# Aqavox — Advanced Chat & Vision Workspace 🌌

Aqavox is a clean, hyper-responsive, single-file AI studio deeply inspired by the Gemini interface. This web application provides a unified developer workspace that seamlessly blends contextual multi-model natural language conversation with precision image synthesis pipelines—**zero API keys, mandatory sign-ups, or local machine learning compilation steps required.**

Built dynamically on top of free public engine endpoints for fluid, unconstrained local experimentation.

---

## ✨ Features

* **🎭 Dual Workspace Modes:** Seamlessly toggle between **Text Chat** and dedicated **Vision Mode** via an interactive, ambient-tinted input box with custom dynamic SVG icon states—mimicking premium commercial AI tools.
* **📂 Local Asset Vault Gallery:** Built-in persistence layers track your creative outputs. Access a dedicated sidebar gallery panel showing your generation collection; click any thumbnail to automatically prep your prompt dock for iterative revisions.
* **🎛️ In-Feed Quick Tuning Modifiers:** Fine-tune generated outputs instantly. Every image card embeds context-aware macro utility pills (`📷 Photorealistic`, `🎨 Anime Style`, `🌆 Cyberpunk`, `✏️ Sketch`) to append rich engine descriptors and auto-re-render your concepts on the fly.
* **🪐 Expanded Engine Cluster:** Supports the market's leading open-source model parameters:
  * **Text Processing:** OpenAI GPT-4o, Mistral Large, and Meta Llama 3 frameworks.
  * **Image Synthesis:** FLUX.1 (High Fidelity Master), FLUX Realism, FLUX Anime, and SDXL-Turbo (Instant Lightning Render).
* **📐 Precise Canvas & Seed Controls:** Modify pixel bounds dynamically and pin generation seed configurations down directly from the settings drawer to ensure deep reproducibility across iterative adjustment turns.
* **🧱 Gemini Layout Collapsibility:** Maximize your focus area with a fluid, hardware-accelerated sliding hamburger navigation toggle that smoothly hides or reveals configuration panels across mobile and desktop breakpoints.
* **💾 Persistent Session Storage:** Built completely on native browser `localStorage`. Your full conversational message threads, selected model properties, aspect constraints, and generated visual indices stay securely cached even if you refresh the browser page.
* **📖 Rich Markdown Table Layouts:** Integrated with `marked.js` parsing modules to transform raw AI payload completions into beautifully aligned, readable dark-mode data tables, clean headers, and structural bold styling.

---

## 🛠️ Tech Stack & Architecture

This architecture is deliberately self-contained within standard web baseline runtimes to maintain zero build friction:

* **HTML5 & CSS3 Standard Layouts:** Semantic containers, responsive grid mechanics, CSS variables root configurations, and linear keyframe shimmer loading animations.
* **Vanilla JavaScript (ES6+ Module Operations):** Clean event listeners, dynamic asynchronous pipeline routing, client-side browser caching, and standard `Blob` link object tracking for single-click image downloads.
* **Third-Party CDNs:** Powered natively by `marked.js` for Markdown compilation.

## 🚀 Quick Start

Because this app has zero builds, package JSON scripts, or node modules, deploying or running it locally requires no installation steps:

1. Clone or download this repository.
2. Ensure your workspace script homepage is named exactly `index.html`.
3. Open the file directly inside any modern desktop or mobile web browser—or deploy it globally for free to platforms like GitHub Pages, Vercel, or Netlify in under 60 seconds.
4. Start creating!
